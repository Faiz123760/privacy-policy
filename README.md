# Postizz2 Privacy Policy

A premium, modern privacy policy page for Postizz2, hosted at https://androcoders.in.

## Features
- **Premium Design**: Dark mode with glassmorphism and smooth animations.
- **Responsive**: Works on all screen sizes.
- **Dynamic**: Automatically updates the "Last Updated" date.
- **SEO Optimized**: Includes meta tags and semantic HTML.

## Tech Stack
- HTML5
- Vanilla CSS3
- JavaScript (ES6+)

## Project Structure
- `index.html`: Main structure.
- `style.css`: Custom styling.
- `script.js`: Interactive logic.
- `hero.png`: Custom 3D security illustration.
